# Harubeori

Harubeori is an early-stage open-source Korean digital diary web app designed to help individuals and families record daily routines, reflections, photos, and meaningful memories in a simple date-based interface.

## Overview

Harubeori started as a personal daily routine and reflection tool inspired by morning diary methods such as brain dump, daily priorities, and time boxing.

The name “Harubeori” is based on the Korean words “하루” and “벼리.”  
It means a daily practice of holding the core thread of the day: emptying the mind, setting meaningful priorities, and weaving time into a lived day.

## Core Concept

Harubeori follows three simple steps:

- **비움 / Emptying**: Write down thoughts, emotions, worries, ideas, and notes.
- **세움 / Setting**: Define the three most important priorities of the day.
- **엮음 / Weaving**: Arrange the day through a simple time-based flow.

## Key Features

- Korean-first daily diary interface
- Date-based daily records
- Morning routine structure: 비움 · 세움 · 엮음
- Mobile-friendly writing flow
- Local browser-based storage
- Date-based record review
- Experimental photo memory feature
- Photo memo concept for family and daily memory records
- JSON backup and restore concept for moving records between devices

## Why Harubeori?

Many productivity and diary apps are too complex for everyday users.

Harubeori focuses on a warm, simple, Korean-friendly experience for people who want to record their daily routines, emotional reflections, family memories, and meaningful moments without complicated setup.

The project is especially designed for:

- Korean users who prefer a simple daily writing ritual
- Families who want to keep small daily memories
- Beginner users who want a lightweight digital diary
- Beginner developers who want to study a simple open-source web app

## User Feedback

Harubeori has been shared with a small group of real users for early feedback.

Early feedback included:

- The Korean name and visual design felt warm and memorable.
- Users wanted to save 1–2 photos per day as meaningful daily memories.
- Users wanted to add short notes to each saved photo.
- Mobile writing experience needs continued improvement.

# Contributing to Harubeori

Thank you for your interest in Harubeori.

Harubeori is an early-stage Korean digital diary project.  
The project is designed to help Korean users record daily routines, reflections, family memories, photos, and meaningful moments through a simple date-based interface.

## Who This Project Is For

This project welcomes:

- Korean users who want a simple digital diary
- Beginner developers learning web app structure
- Designers interested in Korean-first user experience
- Contributors interested in journaling, family memory, and personal productivity tools

## Current Contribution Areas

The project currently needs help with:

- Mobile UI improvement
- Photo memory feature
- Photo memo feature
- JSON backup and restore
- Accessibility
- Code refactoring
- Documentation
- Korean and English user guides

## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Open a pull request
5. Explain what you changed and why

## Beginner-Friendly Contributions

Beginner contributors can help by:

- Fixing typos
- Improving README
- Adding screenshots
- Testing the app on mobile
- Reporting layout issues
- Suggesting clearer Korean wording

## Project Direction

Harubeori aims to remain:

- Simple
- Korean-first
- Mobile-friendly
- Beginner-friendly
- Useful for real daily life

## Code of Conduct

Please be respectful and constructive.  
This project is intended to be a warm and helpful open-source space.

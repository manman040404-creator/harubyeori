# 하루벼리 GitHub Pages 배포용 v0.3 사진기록 추가판

추가 기능:
- 기록 탭에서 날짜별 사진 저장
- 날짜별 최대 2장 저장
- 사진 자동 압축 저장
- 사진별 메모 작성/저장
- 기존 v0.1/v0.2 텍스트 기록 자동 이전

주의:
- 사진은 각 사용자 휴대폰/브라우저의 localStorage에 저장됩니다.
- 브라우저 데이터 삭제 또는 앱 삭제 시 사진도 삭제될 수 있습니다.
- 대량 사진 보관용이 아니라 날짜별 대표 장면 1~2장 보관용입니다.
- 텍스트 백업에는 사진이 포함되지 않습니다.

업로드:
이 폴더 안의 파일들을 GitHub 저장소 최상단(root)에 덮어쓰기 업로드하세요.
